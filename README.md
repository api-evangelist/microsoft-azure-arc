# Azure Arc (microsoft-azure-arc)
Azure Arc REST API enables management of servers, Kubernetes clusters, and data services running outside Azure. It supports onboarding hybrid resources, applying Azure policies, enabling monitoring, and extending Azure management capabilities to on-premises and multi-cloud environments.
